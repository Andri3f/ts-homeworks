Singleton через статичне поле. Використовується для створення єдиного доступного об’єкта (наприклад, масиву для зберження елементів + відповідні методи додавнаня, вилучення, фільтрації). Автоматичне формування ID. Статичне поле генерує унікальні ідентифікатори (спочатку 0, потім інкремент).
